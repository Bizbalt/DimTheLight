# DimTheLight - Smarte Straßenbeleuchtung per LoRaWAN

Wie kann Straßenbeleuchtung flexibler, smarter und nachhaltiger gesteuert werden? Der Kommunalservice Jena möchte Straßenlaternen künftig dynamisch dimmen können – etwa bei Baustellen, Veranstaltungen oder in verkehrsberuhigten Bereichen. Bislang ist die Anpassung der Leuchtprofile nur manuell möglich. Gesucht wird deshalb eine kreative IoT-Lösung, die bestehende Infrastruktur intelligent erweitert statt teuer auszutauschen.

**Deine Challenge**

Eure Aufgabe ist es, einen Prototypen für einen LoRaWAN-NFC-Adapter zu entwickeln, der Straßenlaternen aus der Ferne steuern kann. Ziel ist eine funktionierende Demo, bei der Leuchtprofile drahtlos angepasst und über ein einfaches Dashboard gesteuert werden können. Hardware-Tüftler:innen, Maker und IoT-Fans sind hier genau richtig!

# DimTheLight - Smart Street Lighting via LoRaWAN

How can street lighting be controlled in a more flexible, smarter, and sustainable way? The Jena Municipal Services Department wants to be able to dynamically dim streetlights in the future—for example, at construction sites, during events, or in traffic-calmed areas. So far, adjusting lighting profiles has only been possible manually. We are therefore looking for a creative IoT solution that intelligently expands existing infrastructure rather than replacing it at great expense.

**Your Challenge**

Your task is to develop a prototype for a LoRaWAN-NFC adapter that can remotely control streetlights. The goal is a working demo where lighting profiles can be adjusted wirelessly and controlled via a simple dashboard. Hardware tinkerers, makers, and IoT fans are exactly where they belong here!
